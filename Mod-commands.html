Commands Description Usage
Mute Mute a member so they cannot type or speak. =mute @mention {time} {reason}
Warn Returns user information. =warn @mention {reason}
Kick Kicks a member from the server! =kick @mention {reason}
Ban Bans a member from the server! =ban @mention {reason}
Addrole Adds a role to the user! =addrole @mention {role}
Clear Delete messages at once! =clear [count] , =clear [count] [mention]
Createrole Create a Role In The Server! =createrole {rolename}
Clearwarns Clear all warnings for a user at once! =clearwarns @mention
Delwarn Delete a single warning for a member! =delwarn @mention {count}
Mutelogs Get Mutelogs for a user! =mutelogs @mention
Removerole Removes a role from the user! =Removerole @mention {rolename}
Unban Unbans a member from the server! =unban @mention
Unmute Unmute a member so they can type or speak! =unmute @mention
Setnickname Change the nickname of a user! =setnick @mention {newnickname}
Warnings To get information about warnings for a user! =warnings @mention

Manage Roles

Accesable by -

Clear

Delete messages at once!

Usage -

=clear [count] (For deleting recent messages present in the channel) =clear [count] [tag] (For deleting messages of a specific user) =clear [count] invites (For deleting Discord Server's invite links present in the channel) =clear [count] self (For deleting Aqua X Volt's messages) (Note:- You can delete max 100 at once! but unable to delete those messages which are 14 days older or more.)

Accesable by -

Manages Messages

Announce a message

Usage -

=announce here #channel #colourcode message (For here tagged embeded message)

=announce normal #channel #colourcode message (For sending a normal message)

Accesable by -

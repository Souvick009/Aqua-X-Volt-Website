Addrole

Adds a role to the user.

Usage -

=addrole {user-mention} {role/role name}

Accesable by -

Manage Roles

Removerole

Removes a role from the user.

Usage -

=removerole {user} {role} (Note: The role name should be case sensitive)

Accesable by -

Manage Roles

Ban

bans a member from the server!

Usage -

=ban !user/userid} [Reason]

Accesable by -

Ban Members

Clear

Delete messages at once!

Usage -

=clear [count] (For deleting recent messages present in the channel)
=clear [count] [tag] (For deleting messages of a specific user)
=clear [count] invites (For deleting Discord Server's invite links present in the channel)
=clear [count] self (For deleting Aqua X Volt's messages)

Accesable by -

Manage Messages

Announce

Announce a message

Usage -

=announce everyone #channel #colourcode message (For everyone tagged embeded message)
=announce here #channel #colourcode message (For here tagged embeded message)
=announce image #channel #colourcode imagelink message (For image embeded image)
=announce normal #channel #colourcode message (For sending a normal message)

Accesable by -

Manage Messages, Mention Everyone
